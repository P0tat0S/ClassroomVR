The project was done in Unity 2020.3.33f1
To open add project to Unity Hub and open, most of the source code is in ClassroomVR\Assets\Scripts
To run select the scene "ClassroomVR" and click the Play button on the top of Unity. Scene to Run --> main
The VR headset used to test the application was the Oculus Rift S.

The Assets used for the 3D models where taken from the internet:
VR Hands --> https://developer.oculus.com/downloads/package/oculus-hand-models/
3D Model of the classroom (modified in the project) --> https://www.cgtrader.com/free-3d-models/interior/other/modern-classrom

Controls:
Move --> Left Joystick
Turn Camera --> Right Joystick
Draw --> Right Trigger button
Erase --> Right Grip button
